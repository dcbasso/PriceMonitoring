## Project to Monitoring Prices on some Paraguai Stores

It's like a BI project, to capture and organize the information to know the historical prices of some stores in Paraguay (Ciudad del Este)

## Main stack

Kotlin
Liquibase
Docker
Postgresql (Docker)
Adminer (Docker)
SpringBoot
